COMPUTER

INTRODUCTION

A COMPUTER is a device that accepts information (in the form of digitalized data) and manipulates it for some result based on a program, software, or sequence of instructions on how the data is to be processed.

DIVISIONS OF COMPUTER

A computer primarily comprises of software and hardware, describe briefly as follow:

HARDWARE

Computer hardware includes the physical parts of a computer such as central processind unit (CPU), random access memory (RAM), motherboard, computer data storage, graphics card, sound card, and computer case. It includes external devices such as a monitor, mouse, keyboard, and speakers.

SOFTWARE

Software is a set of informations,data or programs use to operate computer and execute specific task. It is the opposite of hardware, which describes the physical aspects of a computer. Software is a generic term used to refer to applications, scripts and programs that run on a device.E
